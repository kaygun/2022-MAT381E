What is a graph

A Mathematician’s Definition

A graph consists of two sets:

  • A set of vertices (nodes)
  • A set of edges (connections)

A Mathematician’s Definition

V = {A, B, C, D, E}

E = {(A,B), (A,C), (B,D), (C,D), (D,E)

An Intuitive Definition

A graph consists of nodes and connections:

 

V = {A, B, C, D, E}

E = {(A,B), (A,C), (B,D), (C,D), (D,E)
